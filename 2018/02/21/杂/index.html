一个俗人。

发表于 2018-02-21

名: Second Try

我们会努力要作一个风雅的人、一个高尚的人，一个脱离了低级趣味的人，
结果还是在最不经意的时候暴露出自己的俗人本质。
我们试图跟着阳春白雪的调子高唱，
脑子里想的却总是阳春面和白雪公主。

名: First Night

我对着你，想说尽一生，却又一言不发。
望你什么都知道，又盼你眼里不曾出现过我。
让我洒脱的喜欢你，和爱没有关系，和你也没有关系。
只是悄悄的。悄悄地。 ​​​

# 杂七杂八
